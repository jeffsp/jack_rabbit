Detailed Description

template<typename T, class Cont = std::vector<T>>
class jack_rabbit::raster< T, Cont >

raster container adapter

This is a simple container that provides 2D subscripting, STL compatibility, and subregion iteration.

Examples:
raster_example0.cc, raster_example1.cc, raster_example2.cc, and test_raster.cc.
